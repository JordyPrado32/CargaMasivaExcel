Actúa como agente local.

Toma la siguiente issue de GitHub marcada como `agent-ready`.

Flujo:
1. Lee la issue.
2. Crea branch.
3. Implementa cambios mínimos.
4. Ejecuta validaciones disponibles.
5. Corrige errores.
6. Crea commit.
7. Abre PR.
8. No hagas merge.
